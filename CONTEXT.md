# 上毛かるたタイピング

上毛かるた44札の読みをローマ字でタイピングするゲーム。本ドキュメントはこのプロジェクト固有の用語集であり、実装の詳細や仕様書ではない。

## Language

### 入力判定

**撥音ルール**:
撥音「ん」のローマ字許容を文脈で決める規則。次の音が母音・や行・な行のとき、および語末では `nn` のみを正解とし、それ以外では `n` / `nn` の双方を許容する。例: `ぐんまけん` → `gunmakenn`（`gunmaken` は不正解）。
_Avoid_: nルール、撥音処理

**部分入力**:
札の読み全体ではなく、設定した許容範囲（先頭からの一定量など）だけを入力対象とする入力モード。プリセットで範囲を切り替えられる。
_Avoid_: 一部入力、部分タイピング

### 表示モード

**ブラインド入力**:
未入力部分を伏せ字（`_`）で隠して打鍵させる表示モード。Enterキーで読み札全文をヒント表示できる。
_Avoid_: 目隠しモード、ブラインドモード

**再開カウントダウン**:
ポーズ解除からゲーム再開までに挟む約3.8秒のカウントダウン演出。
_Avoid_: リスタートカウント、再開タイマー
